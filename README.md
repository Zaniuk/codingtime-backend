# NOTE: this is only a provisory docs, plans to migrate to an OpenAPI-spec later (mid-priority)

## Users:

- /users/
    - return a list of users (admin only)
- /users/:id
    - return a user
- /users (POST)
    - create a user
- /users/:id (PUT)
    - update a user
- /users/:id (DELETE)
    - delete a user
